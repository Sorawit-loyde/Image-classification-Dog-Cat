# Image Classification of Dogs and Cats

This repository utilizes a convolutional neural network to classify images of dogs and cats and recognize which category they belong to. The goal is to achieve high accuracy through various training techniques and optimizations.

1. Clone the repository:
   ```bash
   git clone https://github.com/Sorawit-loyde/Image-classification-Dog-Cat.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Image-classification-Dog-Cat
   ```

- **High Accuracy**: Utilizes advanced machine learning algorithms for image classification.

The dataset consists of labeled images containing dogs and cats, collected from various sources, ensuring diversity and comprehensiveness for better training results.

- **Dataset Link:** [Kaggle Dogs vs Cats Dataset](https://www.kaggle.com/datasets/bhavikjikadara/dog-and-cat-classification-dataset)
